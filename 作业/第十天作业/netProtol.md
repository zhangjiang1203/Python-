### 1977年规定的OSI七层模型

| 层级  |  功能  | 工作协议 |
| :------: | :-------: | :------: |
| 应用层  |  提供给用户发送消息  | http(80),https(443),smtp(25),DNS(53) telnet(23),ftp(21) |
| 传输层  |  提供链接的机制(选的什么快递，是否包赔等特性)  | TCP(可靠传输)、UDP(不可靠) |
| 网络层  |  提供IP报头  | IP协议 |
| 数据链路层  |  提供一个固定的地址MAC地址  | arp协议(将目的IP地址转换成MAC地址)  rarp协议 |
| 物理层  | 传输消息(光缆 光纤 网线)
